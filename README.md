# Vinay Kumar - Portfolio Website

A personal portfolio website to showcase profile, skills, projects, and contact information.

## Live Preview

Open `index.html` in your browser to view the website locally.

## Project Structure

- `index.html` - Main portfolio page
- `css/` - Stylesheets (Bootstrap, animations, custom styles)
- `js/` - JavaScript files for interactions and effects
- `images/` - Portfolio and background images
- `scss/` - Source SCSS files
- `fonts/` - Font and icon assets
- `lib/` - Third-party front-end libraries

## Features

- Responsive portfolio layout
- Hero section with animated role text
- About and skills section
- Project showcase cards
- Contact section with direct links
- Social profile links

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/vinaykumarbharwal/Portfolio-Website.git
```

2. Open the project folder:

```bash
cd Portfolio-Website
```

3. Open `index.html` directly in a browser.

## Deploy to GitHub Pages

1. Push your code to the `main` branch.
2. Go to repository Settings.
3. Open Pages.
4. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: `main` and `/ (root)`
5. Save and wait for deployment.

## Customize Quickly

Edit these sections in `index.html`:

- Hero name and role
- About section profile info
- Project links
- Contact and social links
